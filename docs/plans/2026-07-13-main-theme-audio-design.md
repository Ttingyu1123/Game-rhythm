# 《月影祕律》正式音樂整合設計

## 音訊資料

正式檔案為 `assets/music/main-theme.mp3`。內嵌標題是 `Moonlit Gear Parade`，作者是 `tingyusaiart`。檔案長 76.80 秒，使用 48 kHz 立體聲 MP3，平均位元率約 187 kbps。節拍分析顯示主速度為 130 BPM，半拍相位約 0.024 秒；37.87 秒與 60.71 秒附近有明顯段落強拍。

## 整合方式

Vite 透過 `new URL(..., import.meta.url)` 將 MP3 納入 production build。`AudioFileLoader` 使用 `fetch` 取得檔案，再交給既有的單一 `AudioContext` 解碼。`AudioEngine` 繼續負責倒數、開始、暫停、恢復與音量；它只替換 Buffer 來源，不改變音訊主時鐘。載入、HTTP 或解碼失敗時，遊戲返回選曲並顯示可讀錯誤，不自動播放程式生成測試曲。

## 譜面

曲目資料改名為「月光齒輪巡遊 / Moonlit Gear Parade」，BPM 設為 130，長度設為 76.8 秒，節拍基準設為 0.024 秒。譜面從第八拍開始，讓玩家先聽約 3.7 秒前奏。前段以四分音符建立操作感，中段加入八分音符、交替與少量雙押，約第 82 拍進入第一個明顯強段，第 132 拍附近進入尾段高潮。最後音符會在音檔結束前保留安全尾奏。

## 驗證

單元測試會先驗證檔案載入成功、HTTP 失敗、解碼失敗、曲目資料、譜面時間範圍、雙押與 130 BPM 格線。production build 必須輸出 MP3 資產。瀏覽器驗收會確認載入正式曲目、倒數後時間前進、首次音符能命中、暫停恢復不漂移、歌曲自然結束、紀錄使用新曲目 ID，且 console 與 page error 都是零。
